# 🇪🇬 EGX Live — البورصة المصرية لحظياً

منصة متكاملة لمتابعة أسعار الأسهم المصرية لحظياً مع تحليل ذكاء اصطناعي.

---

## 🏗️ هيكل المشروع

```
egx-live/
├── server.js          ← السيرفر (Node.js على Railway)
├── package.json       ← إعدادات Node.js
├── data.js            ← نسخة احتياطية من بيانات الأسهم
└── public/
    ├── index.html     ← الصفحة الرئيسية (150 سهم)
    ├── stock.html     ← صفحة تحليل كل سهم
    └── data.js        ← بيانات الـ 150 شركة
```

---

## ⚙️ كيف يعمل المشروع

```
TradingView WebSocket
        │
        ▼
   server.js          ← يخزن الأسعار في الذاكرة
   (Railway)
        │
        ▼
  /api/prices         ← الصفحة تطلب كل 10 ثواني
        │
        ▼
   index.html         ← يعرض الأسعار اللحظية
   stock.html         ← تحليل + Claude AI
```

## 📊 إيه اللي بيتحسب تلقائياً من السعر اللحظي

كل الحاجات دي بتتحدث **فوراً** مع كل تغيير في السعر:

| العنصر | كيف بيتحسب |
|---|---|
| **RSI (14)** | من موقع السعر في نطاق 52 أسبوع |
| **MACD** | من التغير اليومي |
| **MA 20 / MA 50** | نسبة من السعر الحالي |
| **Bollinger Bands** | السعر ± 2 × الانحراف المعياري |
| **Stochastic** | موقع السعر من نطاق السنة |
| **الدعم والمقاومة** | Camarilla Levels من السعر |
| **إشارة شراء/بيع/احتفاظ** | pct52 + upside + توصية المحللين |
| **سعر الشراء المقترح** | السعر الحالي × 0.985 |
| **وقف الخسارة** | السعر الحالي × 0.91 |
| **مستوى الثقة** | موقع السعر من نطاق السنة |

## 🚀 النشر على Railway

### المتطلبات
- حساب [GitHub](https://github.com)
- حساب [Railway](https://railway.app)

### خطوات النشر
```
1. ارفع المشروع على GitHub repository
2. افتح railway.app → New Project → Deploy from GitHub
3. اختار الـ repository
4. Railway هيكتشف Node.js تلقائياً ويشغله
5. Settings → Networking → Generate Domain
6. احفظ الرابط مثلاً: https://egx-live-production.up.railway.app
```

### التحقق من الشغل
```
GET /api/health   ← حالة السيرفر وعدد الأسهم
GET /api/prices   ← كل الأسعار
GET /api/price/COMI ← سعر سهم واحد
```

---

## 🔄 API Endpoints

| Endpoint | الوصف |
|---|---|
| `GET /api/health` | حالة السيرفر + عدد الأسهم + الناقصين |
| `GET /api/prices` | كل الـ 150 سهم بأسعارهم |
| `GET /api/price/:ticker` | سعر سهم واحد |
| `GET /` | الصفحة الرئيسية |
| `GET /stock.html?ticker=COMI` | صفحة سهم معين |

## ⚠️ ملاحظات مهمة

- **أوقات التداول:** البورصة المصرية مفتوحة من **10:00 ص إلى 2:30 م** (الأحد - الخميس). خارج أوقات التداول الأسعار ثابتة عند آخر إغلاق.
- **Railway Free Plan:** السيرفر بينام بعد فترة عدم استخدام — أول طلب بعد النوم بياخد ~30 ثانية.
- **TradingView:** البيانات مجانية غير رسمية — نفس البيانات اللي بتشوفها في الشارت.
- **هذا المشروع لأغراض تعليمية فقط وليس نصيحة استثمارية.**
